ReservaPro es una plataforma SaaS (Software as a Service) diseñada para simplificar y automatizar la gestión de citas y servicios para negocios modernos. Desde barberías hasta clínicas médicas, nuestra solución permite a los dueños de negocios enfocarse en lo que realmente importa: **sus clientes**.

## CARACTERISTICAS PRINCIPALES

-   **Calendario Inteligente:** Interfaz intuitiva para gestionar citas por día, semana o mes.
-   **Recordatorios WhatsApp:** Reducción drástica de ausencias mediante notificaciones automáticas.
-   **Gestión de Clientes:** Base de datos centralizada con historial de servicios y preferencias.
-   **Panel de Estadísticas:** Análisis en tiempo real de ingresos, servicios populares y fidelidad.
-   **Multi-sucursal & Equipo:** Gestión centralizada de múltiples locales y asignación de tareas a colaboradores.
-   **Pagos en Línea:** Integración para aceptar pagos anticipados o señas de reserva.
-   **Modo Oscuro Nativo:** Interfaz adaptable para cualquier entorno de trabajo.

## TECNOLOGIAS UTILIZADAS

Este proyecto utiliza el stack tecnológico más robusto y moderno para asegurar escalabilidad y rendimiento:

-   **Backend:** [Laravel 11](https://laravel.com)
-   **Frontend:** [Vue.js 3](https://vuejs.org) con [Inertia.js](https://inertiajs.com)
-   **Estilos:** [Tailwind CSS](https://tailwindcss.com)
-   **Base de Datos:** MySQL / PostgreSQL
-   **Build Tool:** [Vite](https://vitejs.dev)
